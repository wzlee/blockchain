# blockchain

#### 项目介绍
java版区块链项目

#### 软件架构
采用SpringBoot 2.0.3-REALEASE + MongoDB + Redis + Spring Security

#### 安装教程

1. 检出代码：git clone https://gitee.com/wzlee/blockchain.git
2. 运行：mvn spring-boot:run
3. 访问：http://127.0.0.1:8080
4. 登录：用户名：user,密码：security密码见控制台输出
